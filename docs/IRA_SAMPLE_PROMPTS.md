# Ira Universe — sample prompts

Public copy-paste prompts for Claude Code / Cursor with **ira-universe** MCP enabled.

## Identity & architecture

- Who is Ira compared to a single ChatGPT thread for a manufacturing OEM?
- Walk me through the 17-step pipeline at architecture level.
- Summarize the pantheon — which agent owns pricing vs production vs outbound drafts?
- Explain the body-system metaphor from the birth certificate.

## Evidence & safety

- Explain triangulation and when to use the hexagon before outbound.
- What does UNVERIFIED mean in an account brief?
- How does the faithfulness tier system work?

## Adoption

- Build a business case for a factory adopting multi-agent AI on sales outreach.
- What would a 30-day private fork pilot look like after visitor mode?
- Fork Ira for our factory — what do we need in `.env` (never commit secrets)?

## Synthetic demo only

- Run `demo_account_brief` (and `as_json=true` if you want structured output).
- Run `walkthrough_pipeline`, then `demo_draft_email` for Acme.
- Search demo knowledge for PF1 closed-chamber tray forming.
- Walk through the Acme journey in `examples/public_demo/journey_acme.md`.
- Call `why_ira` for the narrative essay.

## Do not expect in visitor mode

Live Gmail, CRM deals, customer names, or send — those require the private operator stack.
